A crawler script for automatic login to TYUT campus network.

# Features
* based on `selenium`
* using `poetry` virtual-env

# install
* run scripts in `install` folder
* create `data.json` at base path and write:
  ```json
  {
	  "username": "xxx",
	  "password": "xxx"
  }
  ```


# startup
run `startup.sh`
